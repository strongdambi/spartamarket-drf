# spartamarket-drf
Django REST Framework를 사용하여 spartamarket 재구성

## 주요 기능
- 사용자 인증 (회원가입, 로그인, 로그아웃)
- 사용자 프로필 관리
- 상품 등록, 조회, 수정, 삭제 기능(CRUD)
- 사용자 간 팔로우 기능
- 상품 좋아요 기능
- 상품 태그 기능
